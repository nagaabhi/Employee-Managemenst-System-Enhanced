* This Project is about employee management system.
* In this project i have managed the employees based up on the full time, part time and contract based employees.
* In this project i have make used of collection frame works like, stack(for undo or redo the employees), hashmap(acting as the database for my project), streams for easy accesing of the data from the hashmap.
* I have added features like
	*  Adding The Employees.
	*  Deleting the employees by id
	*  Sorting the employees based up on there name.
	*  Activating the employees.
	*  Deactivating the employees.
	*  Counting the total number of active or deactive employees.
	*  Updating the data of the employees.
	*  Generating the data based up on the status.
	*  clearing all the employee details.
* In this project i have added the exception handling for better performance of the project with out breaking of the code...
